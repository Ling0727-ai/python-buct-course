# 示例代码说明

本目录包含了北化课程系统 Python SDK 的各种使用示例。

## 📚 示例列表

### 新示例（推荐）

1. **example1_basic_usage.py** - 基础使用
   - 最简单的使用方式
   - 获取待提交作业的课程列表
   - 适合初学者

2. **example2_detailed_homework.py** - 详细作业信息
   - 获取特定课程的所有作业详情
   - 显示作业标题、截止时间、发布人等信息
   - 支持交互式选择课程

3. **example3_time_analysis.py** - 时间分析
   - 获取所有作业的详细信息
   - 计算剩余时间
   - 标记紧急作业（24小时内截止）
   - 提供统计和建议

4. **example4_homework_tasks.py** - 作业任务要求
   - 获取作业的具体任务描述
   - 解析作业要求内容
   - 显示格式化的任务列表

5. **example5_test_management.py** - 测试管理
   - 获取待进行的测试列表
   - 显示测试信息
   - 提供测试链接

6. **README.py** - 示例导航
   - 运行这个文件可以选择运行不同的示例
   - 提供交互式菜单

### 旧示例（兼容性）

- **demo_client.py** - 交互式客户端演示
- **homework_example.py** - 作业获取示例
- **simple_test.py** - 简单测试
- **full_html_test.py** - HTML解析测试
- **time_test.py** - 时间计算测试
- **subject_test.py** - 课程信息测试

## 🚀 快速开始

### 方式一：运行单个示例

```bash
# 运行基础使用示例
python example/example1_basic_usage.py

# 运行时间分析示例
python example/example3_time_analysis.py
```

### 方式二：使用导航菜单

```bash
# 运行示例导航
python example/README.py
```

然后根据菜单提示选择要运行的示例。

## 📖 学习路径

推荐按以下顺序学习：

1. **入门** → `example1_basic_usage.py`
   - 了解如何创建客户端和登录
   - 了解如何获取基本信息

2. **进阶** → `example2_detailed_homework.py`
   - 学习如何获取详细作业信息
   - 了解作业数据结构

3. **高级** → `example3_time_analysis.py`
   - 学习如何进行时间分析
   - 了解批量操作

4. **专项** → `example4_homework_tasks.py` 和 `example5_test_management.py`
   - 学习特定功能的使用

## 💡 使用提示

1. **首次运行**
   - 确保已安装所有依赖: `pip install requests beautifulsoup4 lxml`
   - 准备好学号和密码

2. **网络要求**
   - 需要能够访问北化课程平台（course.buct.edu.cn）
   - 建议在校园网环境下使用

3. **运行时间**
   - `example1` 和 `example2` 运行较快（几秒）
   - `example3` 运行较慢（可能需要几十秒），因为需要获取所有课程的详情

4. **错误处理**
   - 如果遇到登录失败，请检查学号和密码
   - 如果遇到网络错误，请检查网络连接
   - 如果遇到解析错误，可能是页面结构变化，请提交 Issue

## 🔧 自定义示例

你可以基于这些示例创建自己的脚本：

```python
from buct_course import BUCTCourseClient

# 创建客户端
client = BUCTCourseClient("你的学号", "你的密码")

# 登录
if client.login():
    # 在这里编写你的代码
    pending = client.get_pending_homework()
    
    # 处理数据
    for course in pending:
        print(course['course_name'])
```
